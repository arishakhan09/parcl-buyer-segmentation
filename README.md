# AI-Driven Buyer Segmentation & Market Intelligence
**Company:** Parcl Co. Limited | **Program:** Unified Mentor Data Science Internship

### Project Overview
This repository contains a machine-learning-based clustering system and interactive web application designed to discover hidden patterns in real estate buyer behavior. By moving away from generic targeting, this project groups buyers into four distinct profiles (Global Investors, First-Time Buyers, Corporate Buyers, and Luxury Investors) to drive smarter, data-driven marketing strategies and investment profiling.

### Tech Stack
* **Machine Learning & Data Processing:** Python, Scikit-Learn (K-Means & Hierarchical Clustering), Pandas
* **Interactive Web Dashboard:** Streamlit, Plotly 

---

## ⚙️ Installation Guide

To run this project locally on your machine, follow these steps:

**1. Clone the repository**
```bash
git clone https://github.com/arishakhan09/parcl-buyer-segmentation.git
cd parcl-buyer-segmentation
2. Install the required dependencies Ensure you have Python installed. Then, install the necessary libraries using the requirements.txt file:
pip install -r requirements.txt
(Note: The required libraries are streamlit, pandas, plotly, and scikit-learn)
🚀 How to Run the Project
The project is split into two parts: the machine learning data processing pipeline and the live web dashboard.
Step 1: Process the Data and Generate Clusters First, run the machine learning model. This script will read the raw clients.csv dataset, clean the data, apply feature encoding and scaling, run the K-Means clustering algorithm, and generate a new file called clustered_clients.csv.
python model.py
Step 2: Launch the Streamlit Dashboard Once the clustered data is generated, you can start the interactive web application to visualize the buyer segments, geographic analysis, and client directory.
streamlit run app.py
This command will automatically open the live dashboard in your default web browser (usually at http://localhost:8501).
📁 Project Structure
model.py - The data science pipeline (data cleaning, encoding, scaling, and K-Means clustering).
app.py - The Streamlit web application and UI dashboard.
clients.csv - The raw dataset containing Parcl Co. Limited client demographics and acquisition purposes.
clustered_clients.csv - The processed dataset generated by model.py, including the assigned AI clusters.
requirements.txt - The list of Python packages required to run the environment.
Research_Paper.docx - The final documented report covering the Exploratory Data Analysis (EDA), evaluation metrics (Elbow Method & Silhouette Score), and strategic marketing recommendations.
